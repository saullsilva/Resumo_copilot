# Resumo_copilot
## Desafio Dio - Suzano - Python Developer
### Copilot Studio
  Este resumo apresenta uma visão detalhada sobre o ciclo de desenvolvimento de agentes de Inteligência Artificial no Microsoft Copilot Studio, abrangendo desde a configuração do ambiente no ecossistema Microsoft 365 até a criação de agentes personalizados, tanto por modelos predefinidos quanto do zero. O conteúdo abordou a estruturação lógica de fluxos conversacionais e árvores de decisão, a integração com serviços externos e o refinamento da personalização de respostas para maior assertividade. 
  Além disso, explorou-se o funcionamento da IA generativa na criação automatizada de diretrizes complexas e a capacidade de integrar fontes de dados externas, permitindo que o agente utilize sites específicos como base de conhecimento contextualizada.
